# More Array Methods

Here's a quick demo of some more array methods:
```ruby
[1, 2, 3].reverse        #=> [3, 2, 1]
[1, 2, 3].min            #=> 1
[1, 2, 3].max            #=> 3
[9, 4, 7].sort           #=> [4, 7, 9]
[1, 1, 2].uniq           #=> [1, 2]
["e", "a", "r"].shuffle  #~> ["r", "a", "e"]
```
