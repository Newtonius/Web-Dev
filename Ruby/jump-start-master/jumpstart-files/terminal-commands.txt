

$ pwd (Print Working Directory) -> Tells you where you are.
$ cd (Change Directory)
    $ cd folder_name -> navigate into that folder
    $ cd -> navigate to home directory
    $ cd .. -> navigate one folder up.
$ ls -> list all files and folders in the current directory
$ open . -> open navigator window to the current directory
$ mkdir -> create a new folder in the current directory
$ touch filename -> create a new file in the current directory
$ atom . -> open atom in the current directory (if this isn't working open atom click the atom toolbar and install shell commands)
$ ruby scriptname.rb -> run your ruby script in the terminal (ctrl + c to force quit infinite loop)
$ irb -> open irb in terminal (exit to quit irb)
